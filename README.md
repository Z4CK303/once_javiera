### ·Nuevo repocitorio·
-Para crear una nuevo repositorio vamos a la parte superior derecha de la interfas de usuario donde hay un boton que dice -New- 

-Luego de darle clik veras una especie de formulario en el que pondras el nombre del repocitorio, su descripcion, etc

-despues salbra una ultima parte con varias ventanas y una de ellas pone "Start coding with Codespaces" abajo de ella abra otro boton que dice "create a codespace" y ya estara listo tu repocitorio para empezar a programar
